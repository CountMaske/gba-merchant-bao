# 粤港澳商户宝 · 中小商户交易营收提升助手

> 为中小商户提供的轻量级交易辅助系统，覆盖人员管理、商品管理、SKU 管理、用户暂存区等核心业务模块，助力商户提升交易效率与营收转化。

---

## 📖 项目简介

本项目是一套面向中小商户的后端交易辅助系统，旨在帮助商户快速搭建线上化交易管理能力。系统独立完成了后台人员管理、商品与 SKU 管理、用户暂存区等核心 CRUD 模块，集成阿里云 OSS 实现商品图片的远程存储与访问，使用 HttpClient 打通客户端与后台系统的业务请求链路，完成用户端服务对接。同时封装了业务统计与 Excel 导出接口，为管理端提供动态数据支撑。

---

## 🚀 技术栈

| 分类 | 技术 |
|------|------|
| 核心框架 | Spring Boot 3 + SSM（Spring + SpringMVC + MyBatis） |
| 数据库 | MySQL 8.0 + MyBatis |
| 缓存 | Redis（热点数据缓存 + 布隆过滤器） |
| 消息队列 | RabbitMQ（延迟队列处理订单超时） |
| 实时通信 | WebSocket（交易状态主动推送） |
| 存储 | 阿里云 OSS（图片存储） |
| 接口对接 | HttpClient |
| 版本控制 | Git + GitHub |

---

## ⚙️ 核心功能模块

### 1. 后台管理模块
- **人员管理**：员工账号 CRUD、状态管理（启用/禁用）
- **商品管理**：菜品分类、菜品 CRUD、套餐组合管理
- **SKU 管理**：商品 SKU 维度管理（口味、规格等）
- **用户暂存区**：购物车管理（添加、删除、清空）

### 2. 交易链路模块
- **订单管理**：用户下单 → 支付（模拟）→ 订单状态流转
- **订单超时处理**：RabbitMQ 延迟队列准实时释放库存
- **缓存优化**：Redis + 布隆过滤器拦截无效查询，降低数据库压力
- **实时推送**：WebSocket 推送订单状态变更，替代前端轮询

### 3. 数据统计模块
- 营业额统计（按日/周/月）
- 用户增长趋势统计
- 订单数据统计
- 商品销量排名 TOP10
- Excel 报表导出
